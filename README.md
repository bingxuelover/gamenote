# gamenote
Game guide notes

This is a game guide website project.

## Homepage
`index.html` is the homepage of the site and displays a collection of game guides.

## Hooked Guide
The first guide is for the English game *Hooked* and is available at `guides/hooked.html`.

## How to run
Open `index.html` in your browser to view the homepage.

## Contribution
Feel free to add more game guide pages in the `guides/` folder.
